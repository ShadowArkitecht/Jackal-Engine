Detailed Description

Author
Benjamin Carter

The jackal::ContextSettings_t is a simple struct object that is utilised when the window is first initialised to set and store specific values of the application that will persist throughout the application, such as version and stencil bits.

Due to the structs internal use by the window, it is not exposed to the lua scripting interface and no example is provided.

The jackal::Constants class is a wrapper class around several nested classes that is used to represent commonly used values that will not change throughout the course of the application (such as extensions). It is used just to contain these common values within a single file.

Due to the simplicity of the class, an example is not provided. Specific variables of the classes will be exposed for use by the lua scripting interface.

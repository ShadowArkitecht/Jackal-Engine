Detailed Description

Author
Benjamin Carter

jackal::FileSystem is a basic utility class that is used for conditional checks against local or absolute directories, such as file existence and state of file. It is predominantly used by the VirtualFileSystem for checking if the virtual paths are pointing to valid directories or files.

The FileSystem can be utilised by the user and is exposed to the lua scripting interface.

C++ Code example:

jackal::FileSystem system;
if (system.exists("assets/textures/image.png")
{
std::cout << "It exists!" << std::endl;
}

Lua Code example:

local system = FileSystem()
if system:exists("assets/textures/image.png") then
print("It exists!")
end

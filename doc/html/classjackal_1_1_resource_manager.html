Detailed Description

Author
Benjamin Carter

The jackal::ResourceManager is a high-level class that is used to retrieve and store all of the different resources utilised by the application. These resources includes shaders, textures and materials. The ResourceManager contains a number of cache classes for all the mentioned resources.

The ResoureManager is also used to track the time stamps of shaders, this is utilised to constantly hot-swap and reload shaders when the file is changed. This allows for shaders to be edited during runtime.

using namespace jackal;
// Get a shader from the ResourceManager.
Shader* pShader = ResourceManager::getInstance().get<Shader>("assets/shaders/basic.json");
// Bind the shader and render meshes.
Shader::bind(*pShader);
// Render meshes.
Shader::unbind();

jackal::ResourceManager::~ResourceManager ( )

Destructor for the ResourceManager.

The destructor for the ResourceManager destroys all of the resources that are still being retained by the manager. It also joins the file listener thread back to the main thread.

void jackal::ResourceManager::reload ( )

Reloads the needed shaders for additional use.

This method is invoked every frame, the shaders are only updated if the external glsl shaders are altered. This allows for shaders to be altered whilst the application is still functioning.

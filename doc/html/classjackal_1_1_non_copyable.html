Detailed Description

Author
Benjamin Carter

The jackal::NonCopyable is a simple class that is primarily designed to be inherited from. Any object that inherits from this class is prevented from copying or assigning objects to a different instance of that object. The class abstracts and deletes the copy constructor and assignment operators.

Due to the nature of the class and its uses within the application, it is not exposed to the lua scripting interface and is purely for internal use.

Detailed Description

Author
Benjamin Carter

The jackal::FileReader object is a basic wrapper class for loading and manipulating streams of information that is currently contained on disk. The FileReader will automatically provide a number of checks critical to success when loading files within the Jackal Engine.

The class can make use of the VirtualFileSystem, or load information directly from a specified source. Due to the streaming nature, it is not exposed to the lua scripting interface.

C++ Code example:

// Create a new FileReader object and check it read the file correctly.
jackal::FileReader reader;
// The reader can use the virtual file system.
if (!reader.read("~data/file.txt"))
{
std::cout << "File failed to load!" << std::endl;
}

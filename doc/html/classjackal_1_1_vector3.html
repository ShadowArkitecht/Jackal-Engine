Detailed Description

template<typename T>
class jackal::Vector3< T >

Author
Benjamin Carter

jackal::Vector3 is a templated class responsible for all Vector based mathematics in 3 dimensions. It is commonly used throughout the engine for a variety of different tasks, such as placement of game objects and vertex manipulation.

jackal::Vector3 also contains the most common mathematics and algorithms associated with vectors, such as the dot product and normalisation. As the mathematics is used commonly throughout the application, it is also exposed to the lua scripting interface.

C++ Code example:

using namespace jackal;
// Create two vectors to calculate the cross product.
Vector3f v1 = Vector3f::up();
Vector3f v2 = Vector3f::right();
// Calculate the cross product.
float cross = Vector3f::cross(up, right);

Lua Code example:

-- Create two vectors to calculate the cross product.
local v1 = Vector3.up()
local v2 = Vector3.right()
// Calculate the dot product.
local dot = Vector3.dot(up, right)
